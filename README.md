# MyCoin

MyCoin - веб-приложение для фиксации и анализа личного бюджета

___

## Стэк технологий:
- Backend: Java 17, Java Servlets, PostgreSQL
- Frontend: JSTL, CSS, HTML, JavaScript
- Сборка: Maven

___

## Функционал:
-  Авторизация и регистрация пользователей
- Создание категори доходов/расходов, целей накопления
- Возможность добавлять траты, заработок в категории
- Распределять суммы с заработка на цели накопления
- Просматривать аналитику по периодам: день, неделя, месяц, год
- Просмотреть прогноз по тратам/заработку на будущие месяцы

___

## Установка и запуск
1. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/ReZorDos/MyCoin.git
   cd MyCoin
   ```
2. Создать базу данных и использовать файл `create-database.sql` для создания таблиц БД

3. Создать файл `app.properties` в _src/main/resources/_ и заполнить его по примеру `app.properties.origin`

4. Подключить Tomcat не ниже 10 версии

5. Запусить проект
